Audio Analysis Objects

  • AudioAnalysis

    Describes a tracks structure and musical content, including rhythm, pitch, and timbre.

    All information is precise to the audio sample. Many elements of analysis include confidence values, a floating-point number ranging from 0.0 to 1.0. Confidence indicates the reliability of its corresponding attribute. Elements carrying a small confidence value should be considered speculative. There may not be sufficient data in the audio to compute the attribute with high certainty.

    Read more at the Spotify web API reference.

    Sections are defined by large variations in rhythm or timbre, e.g. chorus, verse, bridge, guitar solo, etc.
