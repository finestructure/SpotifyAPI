SpotifyIdentifier

public struct SpotifyIdentifier : Codable, Hashable, SpotifyURIConvertible

Encapsulates the various formats that Spotify uses to uniquely identify content such as artists, tracks, and playlists. See spotify URIs and ids.

You can pass an instance of this struct into any method that accepts a SpotifyURIConvertible type.

This struct provides a convenient way to convert between the different formats, which include the id, the URI, and the URL.

  • init(uri:ensureCategoryMatches:)

    Creates an instance from a URI. See spotify URIs and ids.

    Uses the following regular expression to parse the id and id categories, in that order:

    "spotify:([a-zA-Z]+):([0-9a-zA-Z]+)"

    The id category must be one of the following, or an error will be thrown:

    • artist
    • album
    • track
    • playlist
    • show
    • episode
    • local
    • user
    • genre

    • unknown

    Throws

    If categories is not nil and the id category of the URI does not match one the required categories or if an id or id category could not be parsed from the URI.
